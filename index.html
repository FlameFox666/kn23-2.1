Таблиця з інформацією:

Тег Опис
<html> Кореневий тег для HTML-документу. Є контейнером для всіх ішних HTML-тегів.
<head> Тег-контейнер для підгрузки метаданих HTML-документу. Метадані встановлює назву файлу, стилі, скрипти та інші метадані.
<meta> Тег задає усі метадані про HTML-документ. Метадані використовуються браузерами, пошуковими системами та іншими веб-сервісами.
<title> Тег, що відображає назву документу на вкладці браузеру. Повинен бути лише текстовим.
<link> Тег визначає зв'язок з документом та зовнішнім ресурсом. Найчастище використовується для зв'язування зі зовнішними таблицями стилів або додавання фавікону (значка сайту).
<style> Тег для CSS стилів документа. Позначається як повинен відображатися кожен елемент сайту.
<body> Тег-контейнер для всього: тексту, заголовків, абзаців, зображень, гіпер-посилань, списків та іншого.
<div> Тег для поділу елементів на розділу/папки. Можна використовувати тег для стилізації через CSS або для використанням JavaScript.
<h1> - <h3> Тег для виділення заголовків. h1 — найважливіший, а чим більше число — тим менш важливіший.
<b> Тег, що робить текст жирним.
<i> Тег, що робить текст курсивним.
<u> Тег, що робить текст підкрессленим.
<p> Тег, що виділяє параграф тексту. Браузери завжди добавляють пусту строку під параграфом.
<ul> Тег для не-організованного списку. ( • ○ ■ )
<li> Тег, що відображає елемент списку. Викорисовується у <ul> та <ol>.
<a> Тег, що використовується для гіпер-посилань.
<table> Тег-контейнер для утворення HTML-таблиці.
<thead> Тег використовується для групування заголовків у HTML-таблиці.
<tbody> Тег використовується для групування основного вмісту HTML-таблиці.
<tfoot> Тег використовується для групування вмісту нижньої частини в HTML-таблиці.
<tr> Тег визначає рядок у HTML-таблиці.
<th> Тег визначає заголовок у HTML-таблиці.
<td> Тег визначає дані у HTML-таблиці.
